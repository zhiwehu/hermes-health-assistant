# 🌿 Hermes Health Assistant

> 把 Hermes Agent 变成你的专属健康养生管家

基于真实使用场景打磨的开源配置方案 —— 不依赖第三方云服务，**100% 本地数据掌控**。

## ✨ 核心能力

### 🗣️ 多模态对话记录
发张早餐照片、说句"今天跑了5公里"、拍个体重秤截图 —— AI 自动解析并结构化记录。

| 输入方式 | 示例 | AI 自动提取 |
|----------|------|-------------|
| 📸 图片 | 智能秤截图 | 体重/体脂/BMI/心率 |
| 📸 图片 | 睡眠监测图 | 时长/深睡/REM/清醒 |
| 🎤 语音 | "中午吃了牛肉面" | 食物类型 + 热量估算 |
| 💬 文字 | "喝水200ml" | 累计进度追踪 |

### 📊 0-100 每日健康打分
不是简单的"打卡"，而是有**加减分机制**的专业复盘：
- ✅ 喝水达标 (+10) | 168严格遵守 (+15) | 睡眠>7h (+10)
- ❌ 超时进食 (-15) | 运动缺失 (-20) | 高盐高脂 (-10)

### ⏰ 智能定时提醒
| 时间 | 内容 |
|------|------|
| 🌅 06:00 | 天气 + 昨日睡眠复盘 + 今日运动/饮食建议 |
| 🌆 22:00 | 今日数据汇总 + 引导填写日记 |
| 📈 周日 20:00 | 周报：一周趋势分析 + 下周计划 |

### 🍽️ 168 间歇性禁食支持
自定义进食窗口（如 08:00-16:00），超时自动标记警告。

### 💊 特殊体质定制
- 结石体质：自动避开高草酸食物（菠菜/酸菜），提醒多喝水
- 脱发治疗：记录用药时间，提醒坚持疗程
- 减脂/增肌：个性化热量与营养建议

## 🚀 快速开始

### 前置要求
- 已安装 [Hermes Agent](https://github.com/yourusername/hermes)
- 本地 Python 3.8+

### 一键安装
```bash
# 克隆仓库
git clone https://github.com/zhiwehu/hermes-health-assistant.git
cd hermes-health-assistant

# 运行安装脚本
chmod +x install.sh
./install.sh
```

安装脚本会自动：
1. 复制 `SKILL.md` 到你的 skills 目录
2. 配置 3 个定时任务（早安/总结/周报）
3. 初始化健康数据目录结构

### 首次引导
安装完成后，和 Agent 说"**开始健康助手配置**"，它会引导你输入：
- 基础身体数据（身高/体重/年龄）
- 健康目标
- 特殊健康状况
- 提醒偏好

## 📁 数据结构

所有数据以 Markdown 存储在本地，方便备份和长期追踪：

```
~/.hermes/profiles/personal-growth/wiki/health/daily/
├── 2026-05-01.md          # 最终日记（含评分、图片、建议）
├── assets/                # 图片资源
│   ├── img_abc123.jpg     # 体测截图
│   └── img_def456.jpg     # 早餐照片
└── conversations/
    └── 2026-05-01.md      # 对话草稿（白天实时记录）
```

## 🎯 进阶功能

### 🔥 热量自动估算
记录饮食时自动估算卡路里：
> "早餐吃了包子和牛奶"
> → **已记录** 🥟 包子(~250kcal) + 牛奶200ml(~130kcal) = **~380kcal**

### 📅 全自动化双层记录模式
- **白天（强制记录）**：所有健康相关对话（哪怕是一句"腿酸"）都会被 AI 实时写入 `conversations/` 草稿文件，绝不遗漏。
- **夜间（无感归档）**：22:00 AI 自动读取草稿，计算热量/得分，**自动保存**为最终日记，仅需用户扫一眼确认。

### 📊 每周健康周报
自动生成包含：
- 体重/体脂趋势折线图
- 168 执行率统计
- 运动总时长对比
- 下周改进建议

## ⚠️ 重要说明

### 医疗免责声明
⚠️ 本助手提供的所有建议仅供参考，**不构成医疗诊断或治疗建议**。如有健康问题，请咨询专业医生。

### 📦 全量数据备份
**包含一切！** 我们的备份脚本不仅备份配置，还备份你的**记忆、收到的图片和文件、对话历史**。

- **备份内容**：
  - 🧠 **记忆与技能** (`memories/`, `skills/`)
  - 📚 **Wiki 与日记** (`wiki/`)
  - 📸 **图片与文件** (`cache/images/` - 包含你发给我的所有原图)
  - 💬 **对话历史** (`sessions/` - 完整聊天记录)
  - ⚙️ **配置** (`.env`, `config.yaml`)

- **执行备份**：
  ```bash
  bash scripts/backup_profile.sh
  ```

- **换机恢复**：
  将备份包解压到新电脑的 `~/.hermes/profiles/` 目录下即可，无需额外操作，Agent 即可恢复所有记忆。

### 隐私与安全
- ✅ 所有数据存储在本地，零云端同步
- ✅ 不上传任何健康数据到第三方
- ✅ 支持完全离线运行（配合本地 LLM）

### 实战避坑
项目中包含详细的 `SKILL.md` 记录了真实使用中的坑：
- 图片日期归属规则
- `patch` 工具处理长文本的陷阱
- Cron Job 消息发送的最佳实践

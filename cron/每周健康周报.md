# 每周健康周报

- **ID**: [替换为实际ID]
- **Schedule**: `0 20 * * 0`
- **Deliver**: [替换为实际channel]

## Prompt

```
你是健康助理。请生成本周健康周报。

## 数据来源
从 `~/.hermes/profiles/[your-profile]/wiki/health/daily/` 目录下读取本周的每日日记文件，汇总分析。

## 需要分析的内容：

### 1. 体重趋势
- 本周体重变化（起始体重 → 结束体重）
- 趋势分析
- 与目标体重的差距

### 2. 运动完成情况
- 完成率统计

### 3. 168进食法执行情况
- 第一餐/最后一餐时间是否达标

### 4. 睡眠质量
- 平均睡眠时长

### 5. 水分摄入
- 达标天数

### 6. 💊 营养素/饮食建议
根据本周情况，给出下周建议：
- 针对特殊体质/慢病
- 针对个人关注点（如头发/睡眠）
- 食补方案

### 7. 总结与建议

## 输出要求：
1. 生成周报发送给用户
2. 将周报保存到 Wiki：`~/.hermes/profiles/[your-profile]/wiki/health/weekly/YYYY-WXX-第X周.md`

## 用户信息模板（安装时替换）：
- [年龄]，[性别]，身高[cm]，当前体重，目标体重
- [特殊体质/病史]
- 168进食窗口：[开始时间]-[结束时间]
```
